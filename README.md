Professional face tracking with OpenCV, MediaPipe & Arduino

## Features
- Real-time face detection
- Arduino stepper control
- PID controller
- Web interface
- Simulation mode
- Video recording

## Quick Start
```bash
pip install -r requirements.txt
python -m src --simulate
```

## Hardware
- Arduino Uno
- NEMA 17 Stepper
- A4988 Driver
- Webcam
